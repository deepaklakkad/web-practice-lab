# 01. Counter App

- **Category:** UI Components / State & Logic
- **Difficulty:** Beginner

## Problem Statement
Create a simple counter application that allows the user to increment, decrement, and reset a number.

## Implementations
- Vanilla JS (Custom CSS)
- Vanilla JS (Bootstrap CSS)
- Vanilla JS (Tailwind CSS)
- React
- Next.js
- Angular

## Core Features
- Display current count
- Increment count
- Decrement count
- Reset count to zero

## Optional Enhancements
- Step value selector (+5, +10)
- Disable decrement below zero

## State Requirements
- count (number)

## Validation Rules
- Count should not go below minimum (if configured)

## Learning Objectives
- Event handling
- State updates
- Conditional UI rendering
